# Shopify (SHOP) Investment Analysis

## Executive Summary

Shopify is the leading e-commerce platform enabling merchants to build, manage, and scale online businesses. The company operates as a "commerce operating system" powering over 10% of all U.S. e-commerce with $292B+ in annual GMV (2024).

### Business Model: Two Revenue Streams

| Segment | % of Revenue | Description | Key Drivers |
|---------|--------------|-------------|-------------|
| **Merchant Solutions** | ~76% | Transaction-based fees on payments, shipping, capital, POS | GMV growth, Shopify Payments penetration (65% of GMV), Shop Pay adoption |
| **Subscription Solutions** | ~24% | Monthly/annual SaaS fees for platform access | Merchant count, plan upgrades, pricing power |

### Key Investment Thesis Points

1. **Platform Moat**: 29% U.S. e-commerce platform market share; network effects from app ecosystem (8,000+ apps)
2. **Payments Flywheel**: Shopify Payments/Shop Pay penetration driving take-rate expansion (~3% of GMV)
3. **B2B Expansion**: B2B GMV growing ~100% YoY - massive untapped market worth trillions
4. **International Growth**: 41% international GMV growth (Q3 2025); Europe +49%
5. **Operating Leverage**: FCF margin expanded to 22% (Q4 2024) after logistics divestiture
6. **AI/Sidekick**: Early mover in AI commerce tools creating merchant stickiness

## Valuation Framework

### How Analysts Value Shopify

| Metric | Current | Historical Range | Peer Comparison |
|--------|---------|------------------|-----------------|
| **P/E Ratio** | ~120x | 50x - 500x | Peers avg: 40x |
| **EV/Revenue** | ~15x | 8x - 50x | SaaS peers: 8-12x |
| **EV/Gross Profit** | ~25x | 15x - 80x | Quality SaaS: 20-30x |
| **PEG Ratio** | ~4x | 2x - 10x | Growth premium |

### Key Valuation Debates

1. **Bull Case (~$200+ PT)**:
   - Commerce TAM is $5T+; Shopify has <1% penetration
   - B2B alone could double revenue over 5 years
   - Operating leverage just beginning (22% FCF margin can expand to 30%+)
   - AI tools create pricing power

2. **Bear Case (~$100 PT)**:
   - 120x P/E assumes perfection
   - Amazon/BigCommerce competition intensifying
   - Take-rate expansion has limits
   - Macro sensitivity (consumer spending)

3. **Consensus View (~$165-180 PT)**:
   - Growth re-acceleration validated
   - Premium warranted but stretched near-term
   - Earnings beats needed to sustain multiple

## Competitive Landscape

### E-Commerce Platform Market Share (2024)

| Platform | U.S. Share | Global Share | Key Differentiator |
|----------|------------|--------------|-------------------|
| **Shopify** | 29% | 19% | Full-stack SaaS, payments, brand control |
| **WooCommerce** | 20% | 24% | Open-source, WordPress integration |
| **Wix** | 18% | 12% | SMB simplicity, website-first |
| **Squarespace** | 15% | 8% | Design-focused, templates |
| **BigCommerce** | 3% | 2% | B2B focus, enterprise features |

### Competitive Moats

1. **vs. Amazon**: "Arming the rebels" - merchants own customer relationships vs. renting them
2. **vs. WooCommerce**: Fully managed vs. self-hosted; less technical expertise needed
3. **vs. BigCommerce**: Larger ecosystem, better SMB onramp, Shop Pay network effects
4. **vs. Legacy (Magento)**: Cloud-native, faster innovation cycles, lower TCO

## Setup Instructions

### 1. Install Dependencies
```bash
pip install marimo yfinance pandas plotly fredapi requests python-dotenv
```

### 2. Configure API Keys (Optional)
Create a `.env` file in this directory:
```env
# Free APIs
FRED_API_KEY=your_fred_key_here          # Get from: https://fred.stlouisfed.org/docs/api/api_key.html

# Premium APIs (optional)
NEWSAPI_KEY=your_newsapi_key             # Get from: https://newsapi.org
POLYGON_API_KEY=your_polygon_key         # Get from: https://polygon.io
ALPHA_VANTAGE_KEY=your_av_key            # Get from: https://www.alphavantage.co
FINNHUB_API_KEY=your_finnhub_key         # Get from: https://finnhub.io
```

### 3. Run the Dashboard
```bash
marimo run shopify_dashboard.py
```

Or edit interactively:
```bash
marimo edit shopify_dashboard.py
```

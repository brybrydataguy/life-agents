Create a new daily journal entry from the template.

1. Read the template from `journals/DAILY_TEMPLATE.md`

2. Create a new file at `journals/daily/{{DATE}}.md` where {{DATE}} is today's date in YYYY-MM-DD format

3. Write the template content to the new file

4. If a file for today already exists, inform the user and ask if they want to overwrite it

5. After creating the file, confirm the creation and show the file path
